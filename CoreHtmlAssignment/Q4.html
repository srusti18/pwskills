Ordered List Of Tags

  1. a tag: The anchor tag is commonly used in HTML to create hyperlinks
  2. h1 tag: Heading tag and it is used to create heading of level 1
  3. img tag: image tag is used to render image on the webpage
  4. sup tag: Super tag is used to write the superscripted text
  5. iframe tag: The iframe tag is used in HTML to embed another HTML document within the current document. It's often used to display content from another source, such as a video, map, or external webpage.
